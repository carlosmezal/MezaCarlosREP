Reto Semana 03 - Analizador de Ventas

Descripción

Programa que procesa un archivo CSV de ventas desde stdin y genera un reporte consolidado por producto.

Métricas calculadas

- Unidades vendidas
- Ingreso total
- Precio promedio

Ordenamiento

Los productos se ordenan por ingreso total de forma descendente.

Ejecución

```bash
python main.py < tests/entrada1.txt
